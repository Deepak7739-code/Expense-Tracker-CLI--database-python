# Expense Tracker CLI 💰🐍

A powerful command-line Expense Tracker built using **Python**, **SQLite**, **argparse**, and **logging**.
This project was developed as part of **Python Programming Internship – Task 5**.

---

# 🚀 Features

✅ Add expenses
✅ List all expenses
✅ Filter expenses by month
✅ Generate reports by category or month
✅ Delete expenses
✅ SQLite database integration
✅ Logging system for debugging & tracking
✅ Friendly CLI interface using `argparse`
✅ Input validation & error handling

# 📌 Commands

---

## ➕ Add Expense

```bash
python task5.py add --desc "Coffee" --amount 120 --category Food
```

### Example Output

```text
✅ Expense added successfully!
```

---

## 📋 List All Expenses

```bash
python task5.py list
```

### Example Output

```text
ID   DATE         DESCRIPTION         AMOUNT     CATEGORY
----------------------------------------------------------
1    2026-05-09   Coffee              ₹120.00    Food
2    2026-05-09   Uber Ride           ₹250.00    Travel
----------------------------------------------------------
Total Expenses: ₹370.00
```

---

## 📅 Filter Expenses by Month

```bash
python task5.py list --month 2026-05
```

---

## 📊 Generate Report by Category

```bash
python task5.py report --by category
```

### Example Output

```text
Food           ₹1200.00
Travel         ₹800.00
Shopping       ₹3000.00
```

---

## 📈 Generate Report by Month

```bash
python task5.py report --by month
```

---

## 🗑 Delete Expense

```bash
python task5.py delete --id 1
```

### Example Output

```text
🗑️ Expense deleted successfully!
```

# 🧠 Database Schema

SQLite database file:

```text

```

Table Structure:

| Column      | Type    |
| ----------- | ------- |
| id          | INTEGER |

| description | TEXT    |
